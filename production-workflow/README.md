# Production Workflow

This guide goes over a production organization and workflow for ingesting media from the camera, editing, and exporting.

## Requirements
* Cloud Storage (Like Box or Google Drive)
* Hedge
* DaVinci Resolve
* Blackmagic Proxy Generator
